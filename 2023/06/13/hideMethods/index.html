注：参考 木易杨前端进阶

opacity: 0 visibility: hidden display: none
结构 占据空间，能绑定点击事件 占据空间，不能绑定点击事件 不占据空间，不能绑定点击事件
继承 非继承属性（子孙节无法修改自身属性显示） 继承属性（子孙节点可以修改自身属性显示） 非继承属性（子孙节点无法修改自身属性显示）
性能 只造成重绘，性能消耗较少 只造成重绘，性能消耗较少 会造成回流，性能消耗较大

补充：

  • 株连性：如果祖先元素遭遇某祸害，则其子孙孙无一例外也要遭殃，比如 opacity: 0 和 display: none，若父节点元素应用了 opacity: 0 和 display: none，无论其子孙元素如何挣扎都不会再出现在大众视野；而若父节点元素应用 visibility: hidden，子孙元素应用 visibility: visible，那么其就会毫无意外的显现出来
  • visibility 会立即显示，隐藏时会延时；opacity 可以延时显示和隐藏
  • opacity 重绘：在一般情况下，opacity 会触发重绘，即 Recalculate style => Update Layer Tree。不管你是否开启 GPU 提升为合成层与否；如果利用 animation 动画，对 opacity 做变化（animation 会默认触发 GPU 加速），则只会触发 GPU 层面的 composite，不会触发重绘
  • opacity 使用场景：自定义图片上传按钮
